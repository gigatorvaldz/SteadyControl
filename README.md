# SteadyControl
Docker | Nest JS | Next JS | MongoDB
---

Запуск:

### Docker
```
docker-compose up
```

По умолчанию:

Client: [localhost:3000](http://localhost:3000/)
Server: [localhost:5000](http://localhost:5000/)

### Вручную

Nest JS
```
npm install

npm run start:dev
```
Необходимо заменить:
В файле env DB_URL на актуальный.

Next JS

```
npm install

npm run dev
```
Заменить API_URL на актуальный для http utils;


По умолчанию:

Client: [localhost:3000](http://localhost:3000/)
Server: [localhost:5000](http://localhost:5000/)
